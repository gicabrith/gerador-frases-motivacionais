
# Gerador de Frases Motivacionais

Projeto simples em Python que gera frases motivacionais aleatórias no terminal.

## Como funciona
O usuário escolhe uma opção no menu e o programa retorna uma frase aleatória usando a biblioteca random.

## Tecnologias usadas
- Python
- Biblioteca random

## Como executar
No terminal digite:

python main.py
